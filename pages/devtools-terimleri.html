Kategori

DevTools Terimleri

Elements

Tarayıcı DevTools içinde HTML yapısını incelemek ve geçici olarak düzenlemek için kullanılan paneldir.

Styles

Seçili elemente uygulanan CSS kurallarını görüntülemeyi ve geçici olarak değiştirmeyi sağlar.

Computed

Seçili elementin tarayıcı tarafından hesaplanmış son CSS değerlerini gösteren bölümdür.

Console

JavaScript kodu çalıştırmak, hata mesajlarını görmek ve log çıktılarını incelemek için kullanılan paneldir.

Network

Sayfanın yaptığı istekleri, yüklenen dosyaları, durum kodlarını ve yanıt sürelerini izlemeyi sağlar.

Lighthouse

Performans, erişilebilirlik, SEO ve iyi uygulamalar için web sayfasını analiz eden raporlama aracıdır.

Sources

JavaScript dosyalarını incelemek, hata ayıklamak ve breakpoint eklemek için kullanılan paneldir.
