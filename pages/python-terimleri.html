Kategori

Python Terimleri

print

Bir değeri veya metni ekrana yazdırmak için kullanılan temel Python fonksiyonudur.

input

Kullanıcıdan klavye aracılığıyla veri almak için kullanılan Python fonksiyonudur.

list

Birden fazla değeri sıralı ve değiştirilebilir şekilde saklayan Python veri yapısıdır.

tuple

Birden fazla değeri sıralı ama değiştirilemez şekilde saklayan Python veri yapısıdır.

dictionary

Anahtar-değer mantığıyla çalışan veri saklama yapısıdır.

function

Belirli bir görevi yapan ve gerektiğinde tekrar çağrılabilen kod bloğudur.

import

Başka bir modül veya kütüphanedeki kodları mevcut Python dosyasına dahil etmek için kullanılır.

Path

Dosya ve klasör yollarıyla daha okunur şekilde çalışmayı sağlayan pathlib sınıfıdır.

json

JSON verilerini okumak, yazmak ve Python veri yapılarına dönüştürmek için kullanılan modüldür.

try

Hata oluşturabilecek kodları güvenli şekilde çalıştırmak için kullanılan hata yakalama bloğunun başlangıcıdır.

except

try bloğunda oluşan hatayı yakalayıp uygun kodu çalıştırmak için kullanılır.

class

Nesne tabanlı programlamada özellik ve davranışları bir araya getiren yapı tanımıdır.

venv

Projeye özel bağımsız Python çalışma ortamı oluşturmak için kullanılan sanal ortam aracıdır.

pip

Python paketlerini yüklemek, güncellemek ve kaldırmak için kullanılan paket yöneticisidir.

API

Farklı yazılımların belirli kurallar üzerinden birbiriyle veri alışverişi yapmasını sağlayan arayüzdür.
