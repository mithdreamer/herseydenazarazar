CSS Terimleri

display

Bir HTML elementinin sayfada nasıl yerleşeceğini belirleyen temel CSS özelliğidir.

flex

Elemanların yatay veya dikey eksende kolayca hizalanmasını sağlayan CSS yerleşim sistemidir.

grid

İçerikleri satır ve sütunlardan oluşan iki boyutlu bir düzende yerleştirmeyi sağlar.

position

Bir elementin normal akışa, ekrana veya başka bir elemana göre nasıl konumlanacağını belirler.

margin

Bir elementin dışındaki boşluğu ayarlayarak diğer elementlerle arasındaki mesafeyi belirler.

padding

Bir elementin içeriği ile kenarlığı arasındaki iç boşluğu belirleyen CSS özelliğidir.

gap

Flex veya grid içindeki öğeler arasında düzenli boşluk bırakmak için kullanılır.

justify-content

Flex veya grid öğelerini ana eksen üzerinde hizalamak ve dağıtmak için kullanılır.

align-items

Flex veya grid öğelerini çapraz eksen üzerinde hizalamak için kullanılan CSS özelliğidir.

var()

CSS değişkenlerinin değerini başka CSS kuralları içinde kullanmayı sağlayan fonksiyondur.

:root

Belgenin en üst düzey elementini seçer ve genellikle genel CSS değişkenlerini tanımlamak için kullanılır.

hover

Kullanıcı bir elementin üzerine geldiğinde uygulanacak stilleri tanımlayan durum seçicisidir.

transition

CSS değerleri değişirken bu değişimin belirli süre ve geçiş efektiyle gerçekleşmesini sağlar.

transform

Bir elementi döndürmek, ölçeklemek, taşımak veya eğmek için kullanılan CSS özelliğidir.

z-index

Üst üste gelen konumlandırılmış elementlerin hangisinin önde görüneceğini belirler.
