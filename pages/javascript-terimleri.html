Kategori

JavaScript Terimleri

const

Yeniden atama yapılamayan bir değişken tanımlamak için kullanılan JavaScript anahtar kelimesidir.

let

Blok kapsamına sahip ve değeri sonradan değiştirilebilen değişken tanımlamak için kullanılır.

querySelector

CSS seçicisi kullanarak HTML içerisindeki bir elementi seçmeye yarayan JavaScript metodudur.

querySelectorAll

CSS seçicisiyle eşleşen tüm elementleri liste benzeri bir yapı olarak seçen JavaScript metodudur.

addEventListener

Bir elementte tıklama, klavye veya form gönderme gibi olayları dinlemek için kullanılan metottur.

if

Belirli bir koşul doğruysa kod bloğunun çalışmasını sağlayan karar yapısıdır.

for

Belirli bir koşul sağlandığı sürece kodu tekrar çalıştırmak için kullanılan döngü yapısıdır.

function

Tekrar kullanılabilir bir kod bloğu tanımlamak için kullanılan JavaScript yapısıdır.

arrow function

Fonksiyonları daha kısa söz dizimiyle yazmayı sağlayan modern JavaScript fonksiyon biçimidir.

fetch

Bir sunucudan veri almak veya sunucuya istek göndermek için kullanılan tarayıcı API'sidir.

async

Bir fonksiyonun asenkron çalışacağını ve Promise döndürebileceğini belirtmek için kullanılır.

await

Asenkron bir işlemin sonucunu beklemek ve kodu daha okunur yazmak için kullanılan anahtar kelimedir.

JSON

Verileri anahtar-değer yapısıyla taşımak için kullanılan hafif ve metin tabanlı veri formatıdır.

DOM

HTML belgesini JavaScript ile okunabilir ve değiştirilebilir bir ağaç yapısı olarak temsil eder.

localStorage

Tarayıcıda küçük verileri kalıcı olarak saklamak için kullanılan yerel depolama alanıdır.
